# 최소화 방식 (케이스 10개 이하)
## 입력
0<=A<=100
0<=B<=100

"통과"==(A+B)>=120
"실패"==(A+B)<120
"과락"==A<50
"과락"==B<50

"다시 입력"==B<0

"다시 입력"==B>100
